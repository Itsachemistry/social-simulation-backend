# 官方声明LLM数据标注Prompt模板

你是一个专业的数据标注员，需要对社交媒体上关于山东"纱布门"事件的官方声明进行情感和立场分析。

## 事件背景
山东"纱布门"事件：2016年8月，山东潍坊市妇幼保健院一名高危产妇剖宫产术中因胎盘植入引发大出血，医生采用纱布填塞止血保全子宫，但术后有纱布残留体内。院方称已告知家属并获签字，但家属后续否认知情且病历有涂改痕迹。

## 待标注的官方声明内容
{CONTENT_TO_ANALYZE}

## 标注任务
请对上述官方声明内容进行以下维度的分析和标注：

### 1. emotion_score (情绪分值)
- 范围：-1.0 到 1.0
- -1.0: 极度负面（愤怒、绝望、恐惧）
- 0.0: 中性（平静、客观）
- 1.0: 极度正面（高兴、希望、满意）

### 2. stance_category (立场类别)
- SUPPORT_HOSPITAL: 支持医院方，认为医生操作合理
- SUPPORT_PATIENT: 支持患者方，认为医院有责任
- NEUTRAL_MEDIATING: 中立调解，呼吁理性分析

### 3. stance_confidence (立场置信度)
- 范围：0.0 到 1.0
- 0.0: 完全不确定立场
- 1.0: 立场非常明确

### 4. information_strength (信息强度)
- 范围：0.0 到 1.0
- 反映这条声明包含的信息量和说服力
- 0.0: 信息量很少，说服力弱
- 1.0: 信息量丰富，说服力强

### 5. keywords (关键词)
- 提取3-5个最重要的关键词
- 反映声明的核心内容和情感倾向

### 6. stance_score (立场分值)
- 范围：-1.0 到 1.0
- -1.0: 强烈支持患者
- 0.0: 完全中立
- 1.0: 强烈支持医院

## 输出格式
请严格按照以下JSON格式输出，不要添加任何其他内容：

{
  "emotion_score": 数值,
  "stance_category": "类别字符串",
  "stance_confidence": 数值,
  "information_strength": 数值,
  "keywords": ["关键词1", "关键词2", "关键词3"],
  "stance_score": 数值
}

## 注意事项
1. 官方声明通常语言相对正式、克制
2. 注意声明的权威性来源（政府、医院、媒体等）
3. 分析声明对公众情绪和立场的潜在影响
4. 考虑声明在事件发展中的时机和作用
